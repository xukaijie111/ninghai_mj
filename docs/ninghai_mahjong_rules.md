# 宁海麻将游戏规则详细说明

## 游戏概述
宁海麻将是流行于中国浙江宁海地区的民间博弈游戏，属于宁波麻将规则的变体之一。游戏使用包含梅、兰、菊、竹、春、夏、秋、冬八个花牌的专用牌具，采用四人对局形式。

## 牌具组成
- **数字牌**: 万子、筒子、索子各1-9，每种4张，共108张
- **字牌**: 东、南、西、北、中、发、白各4张，共28张
- **花牌**: 梅、兰、菊、竹、春、夏、秋、冬各1张，共8张
- **总计**: 144张牌

## 开局规则

### 确定庄家
1. 通过扔二颗骰子得到数目A
2. 以逆时针从扔的人开始数，数到A的人就是庄家
3. 庄家扔骰子，扔出数目B，数B次数出一人再扔
4. 二次扔出的数目相加得出数目C
5. 从第二次扔骰子的玩家前的牌开始数C张牌，从庄家开始抓牌

### 起手牌数
- 庄家：14张牌
- 其他玩家：13张牌

### 补花规则
- 如果有花牌，从庄家开始，按逆时针顺序补花
- 从最后的牌开始补花

### 第一圈特殊规则
四人坐下开始游戏时，第一轮的第一圈：
- 只能打筒子和万子两种牌
- 不能进行吃、碰、杠等操作

## 花牌系统

### 花牌对应关系
- **东风家**: 梅花、春花 - 每张增加一番
- **南风家**: 兰花、夏花 - 每张增加一番
- **西风家**: 菊花、秋花 - 每张增加一番
- **北风家**: 竹花、冬花 - 每张增加一番

### 花牌特殊组合
- 收集四季花（春、夏、秋、冬）：算两番
- 收集四君子（梅、兰、菊、竹）：算两番

## 胡牌规则

### 基本牌型
1. **11，123，123，123，123** - 一对 + 四个顺子
2. **11，123，123，123，111** - 一对 + 三个顺子 + 一个刻子
3. **11，123，123，111，111** - 一对 + 两个顺子 + 两个刻子
4. **11，123，111，111，111** - 一对 + 一个顺子 + 三个刻子
5. **11，111，111，111，111** - 一对 + 四个刻子（对对胡）

### 特殊牌型
1. **清一色**: 全部索子或万子或筒子 - 三番牌
2. **字一色**: 全部为字牌 - 三番牌
3. **混一色**: 字牌加上一色的索子或万子或筒子 - 两番牌
4. **对对胡**: 全部刻子 - 二番牌，如果同时符合清一色要求为三番牌

## 特殊规则

### 流局规则
- 倒数9墩牌时流局
- 最后是单数牌不算
- 每一杠多加一墩

### 三摊规则
如果一家吃碰杠在牌面上有三次，且满足以下条件之一：
- 三次都是同一色
- 两字碰加任何一色
- 一字碰加同一色的两碰吃杠

此时其他三家如果没有听牌的家：
- 不能打与三摊同色的牌，否则要包牌
- 如果有一家牌面上有碰杠三次，其他三家不能打大幺牌，否则要包牌

## 计分系统

### 花牌计分
- 对应位置花牌：每张1番
- 四季花或四君子：2番
- 红中对、发财对、白板对：碰或暗刻或杠为1番
- **三番封顶**（拉子牌）

### 胡数计算

#### 基础胡数
- **花牌**: 春夏秋冬、梅兰菊竹八种花 = 4胡
- **幺九牌碰出**: 一筒、九筒、一万、九万、一索、九索、东南西北风、中发白 = 4胡
- **幺九牌暗刻**: 8胡
- **幺九牌暗杠**: 32胡
- **幺九牌明杠**: 16胡
- **其他牌碰出**: 2胡
- **其他牌暗刻**: 4胡
- **其他牌暗杠**: 16胡
- **其他牌明杠**: 8胡
- **自摸**: 2胡
- **夹中胡**: 2胡（例如听一索和三索中的二索）

### 计分模式

#### 1. 幺半三（30胡底）
- 基础底分：30胡
- 计算公式：(底分 + 胡数) × 番数
- 超一进十，最后胡数为整数
- 三番直接300胡封顶
- 银子计算：10胡=10银，120胡=120银，300胡=300银
- 没胡的三家算法：边家每多10胡收5银，头家收10银

#### 2. 幺半六（60胡底）
- 基础底分：60胡
- 计算公式：(底分 + 胡数) × 番数
- 超一进十，最后胡数为整数
- 三番直接500胡封顶
- 银子计算：0-20胡不加，30-60胡加半，70-100胡进一
- 没胡的三家算法：边家比边家多出50胡收50银，比头家多出30胡收50银

#### 3. 推倒八
- 无番数：100胡
- 一番：200胡
- 二番：400胡
- 三番：800胡
- 银子计算：相应为100银、200银、400银、800银
- 没胡的三家算法：边家比边家多出50胡收50银，比头家多出30胡收50银

#### 4. 幺二六（100胡底）
- 基础底分：100胡
- 计算公式：(底分 + 胡数) × 番数
- 超一进十，最后胡数为整数
- 三番直接1000胡封顶
- 银子计算：0-20胡不加，30-60胡加半，70-100胡进一
- 没胡的三家算法：比头家多出30胡收50银

## 游戏流程

### 1. 准备阶段
- 确定庄家
- 发牌（庄家14张，其他13张）
- 补花

### 2. 游戏阶段
- 庄家首先出牌
- 玩家按逆时针方向依次出牌
- 可以进行吃、碰、杠操作（第一圈除外）
- 检查胡牌条件

### 3. 结算阶段
- 计算胡数和番数
- 根据选择的计分模式结算
- 确定下一局庄家

## 术语解释

- **番**: 倍数单位，影响最终得分
- **胡**: 基础分数单位
- **拉子牌**: 三番封顶的牌
- **三摊**: 一家有三次吃碰杠且符合特定条件
- **包牌**: 违反三摊规则需要承担责任
- **夹中**: 听中间的牌，如听二索时有一索和三索
- **边家**: 上家和下家
- **头家**: 对家
